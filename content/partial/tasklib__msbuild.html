MSBuild Tasks

MsBuildTask

Runs MSBuild.exe with specified arguments. See MSBuild Command-Line reference for configuration details.

Input

  • Project file;
  • MSBuild switches;

Examples

A task that generates a NuGet spec for Rosalia.Core package:

new MsBuildTask<Context>()
    .FillInput(c => new MsBuildInput()
        .WithProjectFile(c.Data.SolutionFile)
        .WithConfiguration(c.Data.Configuration))
